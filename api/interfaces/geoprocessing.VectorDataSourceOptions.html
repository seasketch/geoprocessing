cacheSize

cacheSize: number
  • Defined in src/datasources/VectorDataSource.ts:48

Max number of feature bundles to keep in memory. Calls to .fetch() will not return more than the contents these bundles, so this acts as an effective limit on subsequent analysis.

dissolvedFeatureCacheExcessLimit

dissolvedFeatureCacheExcessLimit: number
  • Defined in src/datasources/VectorDataSource.ts:68

When features are requested by fetch, bundled features with matching union_id will be dissolved into a single feature. This dissolved feature is expensive to create and so may be cached. A cache may contain more bundles than needed, and this variable sets a cap on that number.

default

3
