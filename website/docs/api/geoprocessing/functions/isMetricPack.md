# isMetricPack()

```ts
function isMetricPack(json): json is MetricPack;
```

Checks if object is a MetricPack. Any code inside a block guarded by a conditional call to this function will have type narrowed to MetricPack

## Parameters

| Parameter | Type  |
| --------- | ----- |
| `json`    | `any` |

## Returns

`json is MetricPack`
